001 : SuperHeroes App

Este proyecto fue creado para el desafío React de Alkemy Labs. La aplicación web fue construida para armar un grupo de 6 superhéroes del universo Marvel y DC Comics y poder ver sus estadísticas individuales y como equipo. La app es responsive, efectúa una validación de formularios (Login page), realiza peticiones HTTP a SuperHero API, los datos de inicio de sesión, así como el equipo construido por el usuario, persisten en el almacenamiento local del navegador. Se utilizaron varias librerías: React (interface de usuario), Formik (formularios), Axios (peticiones), react-router (rutas), react-icons y Bootstrap 5 para los estilos. Para más información, entrar al repositorio en GitHub.
